Jobly

Jobly is a full stack job search web application. It has authentication and authorization built in place using JWT tokens and some of the routes are protected with an authentication middleware.

The backend includes a classic RESTful API setup for storing and providing information on companies, jobs, and users, so the same information can be accessed using a tool like Insomnia.

Tech stack used on this project consists of React on the frontend, Node and Express on the backend, and PostgreSQL for database design.

The main focus of this project was building a single page React app using React Hooks, with a fully functioning backend.

Netflix-Clone

Netflix-Clone is a fully functional front-end clone of Netflix web application, a widely popular paid streaming service.

Instead of creating a backend, Firebase Authentication was used for both user authentication and authorization. Similarly, Firebase Cloud Storage was used to solve database needs. This choice was made to get experience with different technologies.

The application also has fuzzy searching functionality using Fuse.js, for live searching and filtering movies and TV shows.

The main focus of this project was building a React clone of Netflix while utilizing compound component pattern for component design, CSS framework Styled-Components for styling the application, and custom hooks for reusing stateful logic.

Blogly

Blogly is a simple but slick looking, full stack, blogging web application. The app supports typical blogging functionalities like creating new blog posts, editing or removing existing blog posts, commenting on blog posts, upvoting and downvoting, and more.

Blogly was built using React on the frontend, Redux and Redux-Thunk for state management, Node and Express on the backend, and postgreSQL for database design.

The main focus of this project was getting more experience with React Hooks by building another React app, but also using Redux and Redux-Thunk for state management instead of using useState or props drilling.
